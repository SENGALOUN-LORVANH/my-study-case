# Desktop Program - C#

## Overview
This project is a desktop application developed in C# using Visual Studio 2022. It connects to an SQL Server database for data management and storage.

## Features
- User Authentication
- Data Management
- Interactive User Interface
- Database Connectivity

- .NET Framework 4.8+
- SQL Server 2019+
